# Chem e-tests site

Статический сайт (HTML/CSS/JS) про электронное тестирование на уроках химии: теория, история, банк тестов, справочник, галерея, контакты.

## Запуск локально

Рекомендуется запускать через локальный сервер, чтобы `fetch()` корректно прочитал `assets/data/tests.json`.

- Python:
  - `python -m http.server 8080`
  - открой `http://localhost:8080`
- Node.js:
  - `npx serve .`

## Где править тесты

Банк тестов: `assets/data/tests.json`.

## Настройки (в шапке)

- Тема (светлая/темная)
- Стиль (минимализм/аура/моно)
- Размер текста (A−/A+)
- Высокий контраст
- Меньше анимаций

## Контакты

Форма использует EmailJS (без своего сервера). Ключи вставляются в `assets/js/email.js`.
