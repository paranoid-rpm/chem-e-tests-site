# Chem e-tests site

Современный статический сайт (HTML/CSS/JS) про электронное тестирование на уроках химии: теория, история, банк тестов, галерея, контакты.

## Запуск локально

### Вариант 1 (рекомендуется): локальный сервер

- Python:
  - `python -m http.server 8080`
  - открой `http://localhost:8080`
- Node.js:
  - `npx serve .`

### Вариант 2: открыть index.html

Иногда браузер блокирует загрузку `assets/data/tests.json` через `fetch()` при открытии файла напрямую, поэтому лучше запускать через сервер.

## Где править тесты

Банк тестов лежит в `assets/data/tests.json`.

## Отправка формы на почту

Используется EmailJS (без своего сервера). Открой `assets/js/email.js` и вставь свои ключи.
