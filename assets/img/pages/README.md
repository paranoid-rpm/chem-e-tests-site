# Фотографии для страниц

Эти фото отображаются прямо в разделах (главная/тесты/теория/…); если файла нет — покажется плейсхолдер.

## Куда класть

`assets/img/pages/`

## Как назвать

- Главная: `index-01.jpg`, `index-02.jpg`, `index-03.jpg`
- Тесты: `tests-01.jpg`
- Теория тестов: `theory-01.jpg`
- История: `history-01.jpg`
- Химия: `chemistry-01.jpg`
- Справочник: `glossary-01.jpg`
- Контакты: `contact-01.jpg`

Можно использовать JPG или WebP, но если меняешь расширение (например `.webp`) — нужно поправить путь в соответствующей странице.
