# Сюда класть фотографии

Положи свои изображения в эту папку и назови их по шаблону:

- `photo-01.jpg`
- `photo-02.jpg`
- ...

Страница `gallery.html` уже настроена так, что если фото ещё нет — покажется плейсхолдер, а как только ты добавишь файл с нужным именем, он подхватится автоматически.

Рекомендации:
- Формат: JPG или WebP.
- Размер: 1600–2400 px по ширине обычно достаточно.
- Вес: старайся держать до ~400–700 KB на фото.
